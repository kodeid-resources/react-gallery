# react-gallery

Example app for React Component styling

## Styling methods

- Inline styles
- Radium
- External CSS
- CSS modules
- Styled Component
- Emotion

## Install

```
git clone git@github.com:kodeid-resources/react-gallery.git
npm install
```

## Run

```
npm start
open http://localhost:3000
```

Includes only dev environment build target.
